3. Placez le capteur à ultrasons

Placez le HC‑SR04 là où vous souhaitez détecter un mouvement. Avertissement: Il ne détectera pas les mouvements éloignés.

4. Connectez votre Arduino à votre PC

Puis ouvrez https://unoserial.oneapp.dev/

5. Cliquez sur « Connecter »

Sélectionner Arduino UNO dans la fenêtre contextuelle.

6. Profitez-en !

Votre système Camcookie Security UNO R3 est désormais actif.
