Camcookie Security – UNO R3

🚧 IN PROGRESS 🚧

The code and functions may not fully work correctly. It may go off randomly.

This currently only works on Microsoft Edge on a PC. It will not work on mobile devices.

1. Upload the Arduino Code

Add the .ino code from https://camcookie876.github.io/original/DOCS/arduino/1/code/code.txt to your Arduino UNO R3.

2. Connect the Wires

🟦 Ultrasonic Sensor (HC‑SR04)

3. Place the Ultrasonic Sensor

Put the HC‑SR04 wherever you want to detect motion. Warning: It will not detect far‑away motion.

4. Connect Your Arduino to Your PC

Then open https://unoserial.oneapp.dev/

5. Click “Connect”

Select Arduino UNO in the popup.

6. Enjoy!

Your Camcookie Security UNO R3 system is now active.
